Adapting Skills to Novel Grasps: A Self-Supervised Approach

If a robot learns a skills with a grasped object (e.g. a tool), then that skill will usually fail if the robot grasps the object in a different way to when it learned the skill. In this work, we introduce a self-supervised data collection method, which enables a robot to adapt a skill to a novel grasp even though the skill was learned using a different grasp.

One-Shot Imitation Learning: A Pose Estimation Perspective

Studied a novel combination of trajectory transfer and unseen object pose estimation, which enables new tasks to be learned from just a single demonstration, without requiring any further data collection or training. Under this formulation, experiments provide a deep dive into the effects of pose estimation errors and camera calibration errors on task success rates.

Learning Eye-in-Hand Camera Calibration from a Single Image

Studied a range of different learning-based methods for extrinsic calibration of a wrist-mounted RGB camera, when given only a single RGB image from that camera. We found that a simple direct regression of calibration parameters performed the best, and also outperformed classical calibration methods based on markers.

Hybrid ICP

ICP algorithms typically involve a fixed choice of data association method and a fixed choice of error metric. In this paper, we propose a novel and flexible ICP variant, which dynamically optimises both the data association method and error metric based on the live image of an object and the current ICP estimate.

Framework and Benchmarks for Combinatorial and Mixed-variable Bayesian Optimization

We introduce a modular framework for Mixed-variable and Combinatorial Bayesian Optimization (MCBO) that addresses the field's lack of systematic benchmarking. The framework enables effortless combination of Bayesian Optimization components and provides diverse benchmarking tasks. We implemented 47 novel MCBO algorithms and benchmarked them against 12 existing solvers across ten tasks in over 4000 experiments. Results reveal a superior combination of MCBO primitives that outperforms existing approaches and demonstrate the importance of model fit and trust regions.

Toward real-world automated antibody design with combinatorial Bayesian optimization

Designing therapeutic antibodies requires optimizing both antigen-binding affinity and developability properties. We frame this as a combinatorial search problem where a black-box oracle simulates binding affinity from antibody-antigen sequence pairs. Since the vast antibody sequence space prevents exhaustive search both computationally and experimentally, we need efficient mechanisms to find optimal antibody sequences that maximize binding affinity while maintaining favorable biophysical properties.
